# E-Office System

React 18 + TypeScript + Tailwind CSS + Vite

## Quick Start
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Features
- Full CRUD (Create, Read, Update, Delete)
- Search & filter
- Status toggle
- Pagination
- Form validation
- Toast notifications
- Responsive design
